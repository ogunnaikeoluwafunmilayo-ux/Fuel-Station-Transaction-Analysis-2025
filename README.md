# Fuel Station Transaction Analysis — 2025 Dashboard

## Project Overview
An end-to-end Excel dashboard analyzing 31,896 fuel station transactions 
across 2025, tracking revenue performance, fuel costs, gross profit, 
payment methods, and attendant performance.

---

## Key Statistics
| Metric | Value |
|---|---|
| Total Transactions | 31,896 |
| Period Covered | January — December 2025 |
| Fuel Types | PMS, AGO |
| Payment Methods | Cash, POS, Transfer |

---

## Tools Used
- Microsoft Excel (Advanced)
- Pivot Tables & Pivot Charts
- Data Cleaning & Transformation
- Dashboard Design & Visualization

## Key Analysis Performed
- Monthly revenue and fuel cost tracking
- Gross profit analysis by month and quarter
- Fuel type performance comparison (PMS vs AGO)
- Payment method breakdown (Cash, POS, Transfer)
- Attendant performance analysis
- Peak transaction time analysis
- Weekday vs weekend sales comparison

---

## Key Insights
- August recorded the highest total revenue
- PMS was the dominant fuel type across all transactions
- Transfer and POS were the most used payment methods
- Morning and afternoon periods recorded highest transaction volumes
